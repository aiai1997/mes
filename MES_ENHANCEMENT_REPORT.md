## 🚀 部署和使用指南

### 1. 环境准备
```bash
# 安装依赖
pnpm install

# 执行数据库迁移
node scripts/run-migration.ts

# 初始化权限数据
node scripts/init-rbac-db.ts
```

### 2. 运行测试
```bash
# 运行系统测试
node scripts/test-rbac.ts
```

### 3. 启动应用
```bash
# 开发模式
pnpm dev

# 生产模式
pnpm build && pnpm start
```

### 4. 权限配置
- 默认创建6个角色（管理员、业务员、跟单员、车间工人、组长、财务）
- 自动分配基础权限
- 支持动态权限调整
